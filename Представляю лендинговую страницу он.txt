Представляю лендинговую страницу онлайн фитнес-клуба.
Для придания смысла и структуры html разметки, я использовала семантические тэги.
Хэдер <header> - навигационная верхняя панель.
Мейн <main> - основное наполнение страницы, состоит из пяти секций <section> - блоков с информацией
Футер <footer> - нижняя панель с информацией о контактах и авторском праве. Откуда можно будет перейти в соцсети клуба.

Хэдер зафиксирован вверху страницы и при прокрутке всегда остается в поле видимости с помощью свойства position: fixed. Пункты панели оформлены ссылками на соответствующие страницы.

В первой секции - мотивирующее фото и приветствие. Для лучшей читабельности приветствия использовано свойство размытия backdrop-filter: blur(3px).

Следующая секция - это собственно предлагаемые абонементы на занятия в клубе. Этот блок информации выполнен с помощью сетки grid, которая позволяет удобно размещать карточки с предложениями.

Блок формы оформления заказа абонемента выполнен с применением элемента html - form. Тут три поля для ввода информации клиентом с клавиатуры, для этого использованы тэги input, а также кнопка для отправки информации - button. При клике в поле ввода информации - он меняет цвет, показывая, что поле активно.

Следующие две секции - это блоки с информацией - комфорт занятий дома и направления тренировок - 
я применила в свойствах css гибкий контейнер flex, с помощью которого легко размещать нужную информацию в доступном для пользователя виде. К картине применила свойства transition & transform: rotate - для того, чтобы при наведении курсора на картинку, она переворачивалась и менялась на другую. 

Для выделения блоков с информацией, применила свойство hover - при наведении в блоках секции комфорт - увеличивается текст, а в секции направления - увеличивается сама карточка с информацией.

В нижнем блоке - футере - также применен гибкий контейнер flex для выстраивания информации по столбцам. Значки соцсетей при наведении на них курсора становятся цветными.

Также предусмотрена закрепленная кнопка для быстрого заказа - "Начать заниматься". Для реализации такой кнопки использован стиль animation со свойством infinite, а также @keyframes правило для пульсации кнопки.
 И ярлык-закладка для доступа к контактам. Чтобы изначально был виден только ярлычок - блоку заданы параметры с отрицательным внешним отступом margin-left. А при наведении на него курсора, свойством hover, значение отступа отрегулированы, чтобы стал виден весь блок.

Для корректного отображения страницы на экранах разной ширины применена адаптивная верстка с помощью медиа-запросов - media-scring и под разную ширину экрана - отредактированы свойства элементов.
